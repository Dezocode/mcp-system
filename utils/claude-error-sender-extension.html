Claude Error Log Sender

Automatically format and send error logs to Claude for analysis

How to use:

  1. Paste your error log in the text area below
  2. Click "Format & Copy" to prepare the message
  3. Go to Claude and paste (Ctrl/Cmd+V) in the message box
  4. Or use "Send to Claude Tab" if Claude is open in another tab
